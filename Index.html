React Native

React Native é um framework JavaScript criado pela Meta. Ele é usado para desenvolver aplicativos mobile para Android e iOS de forma nativa. Sua principal vantagem é permitir o reaproveitamento de grande parte do código entre as plataformas, tornando o desenvolvimento mais rápido e prático.

Expo

Expo é um conjunto de ferramentas baseado no React Native que facilita a criação de aplicativos mobile. Ele oferece um ambiente já configurado, recursos prontos e a possibilidade de testar o aplicativo no celular de forma rápida, sem precisar mexer diretamente no código nativo.
